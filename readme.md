# quizbuilder

## installation
```sh
$ npm install
$ cd client && npm install
$ cd ..
```
## running the app
### 1: local mongodb server
```sh
$ mongod
```
### 2a: concurrently running server and api
```sh
$ npm run dev
```
### 2b: alternatively in 2 seperate terminals
```sh
$ nodemon server.js
$ cd client && yarn start
```
